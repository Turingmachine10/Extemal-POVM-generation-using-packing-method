The program implements the algorithm in the paper "EFFECTIVE METHODS FOR CONSTRUCTING
EXTREME QUANTUM OBSERVABLES"[Published paper](https://iopscience.iop.org/article/10.1088/1751-8121/ab8d52),[Arxive print](https://arxiv.org/pdf/1809.09935) by Dr Erkka Haapasalo and Dr Juha-Pekka Pellonpää.
